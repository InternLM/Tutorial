# 茴香豆：企业级知识问答工具实践闯关任务

## 基础任务（完成此任务即完成闯关）

- 在 InternStudio 中利用 Internlm2-7b 搭建标准版茴香豆知识助手，并使用 Gradio 界面完成 2 轮问答（问题不可与教程重复，作业截图需包括 gradio 界面问题和茴香豆回答）。知识库可根据根据自己工作、学习或感兴趣的内容调整，如金融、医疗、法律、音乐、动漫等（优秀学员必做）。
> 如果问答效果不理想，尝试调整正反例。

## 挑战任务
