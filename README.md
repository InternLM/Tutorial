# 书生·浦语大模型实战营第二期

![](./asset/camp.png)

为了推动大模型在更多行业落地开花，让开发者们更高效的学习大模型的开发与应用，上海人工智能实验室重磅推出书生·浦语大模型实战营，为广大开发者搭建大模型学习和实践开发的平台，两周时间带你玩转大模型微调、部署与评测全链路。

第二期大模型实战营正在火热筹备中，预期3月份启动，欢迎社区小伙伴们参与到课程制作中。
